# Github Finder Application - React

 - Github Finder App allows you to search for a user and fetch relevant user profile.
 - Provides an option to visit the searched user profile to get the profile information and a list of repositories.
 - Uses Github API to make requests and fetch the reponse.

# Usage:
- Clone the repo
```git clone https://github.com/skrish35/github-finder-app.git```

- Install Dependencies
```npm install```

- Run the application
```npm start```
- Open http://localhost:3000

- Builds the app for production to the build folder
```npm run build```
